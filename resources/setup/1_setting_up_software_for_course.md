<center><h2>Install Anaconda</h2></center>

- Follow these directions for macOS __graphical__ install [docs.anaconda.com/anaconda/install/mac-os](https://docs.anaconda.com/anaconda/install/mac-os)
- Python 3.7 version
- Skip PyCharm installation

<center><h2>Download GitHub Repository</h2></center>

Use GitHub like Dropbox for now, except it will require manual syncing.

You'll just download files that I post. Later we'll use it how a professional coder use it.

Setting-up GitHub
-------

1. Open a web browser to http://www.github.com/brianspiering/
2. Find website for course repository.
3. Click on green code button and select "Open with GitHub Desktop Option"
4. Install [GitHub Desktop](https://desktop.github.com/) if prompted
5. If you don't how to navigate the file system, put everything on desktop.


<center><h2>Install Conda Environment</h2></center>

1. Launch Anaconda Navigator
2. Click on Environments
3. Click on Import
4. Click on folder icon and navigate to your local copy of the GitHub repository for the course.

<center><h2>Running Jupyter Notebook</h2></center>

- Find Anaconda Navigator
- Launch Jupyter Notebook
